Newtflix

Newtflix is a replica of netflix built from the ground up using pure React and Ruby on Rails, with videos and images hosted on AWS. The project was built to emulate Netflix's dynamic frontend while attempting to meet the challenge of minimizing backend calls for video fetching. Check it out here. Or to see the code click here.

Just Roll

Just Roll is a 3D javascript based game played entirely in the user's browser. The player can interact with movable 3D object using simulated physics and collision detection. Try to get to the goal without falling off the level! check it out here. Or check out the repository here.

Thursday the Fourteenth

Thursday the Fourteenth is a primitive, augmented reality, tag game. Players use wrist mounted sensors to tag enemies and deal damage. One set of sensors read gestures using 9DOF sensors. The other uses a webcam to detect colored spheres on enemy players. Check out the ui code here here. Check out the server code here
